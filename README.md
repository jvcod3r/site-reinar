# 🏠 Reinar — Site Institucional

Site institucional desenvolvido para a **Reinar**, empresa de restauração, limpeza e polimento de pisos localizada no Itaim Paulista, São Paulo - SP.

---

## 📋 Sobre o Projeto

Landing page completa com foco em apresentação de serviços, galeria de fotos, depoimentos de clientes e informações de contato. O layout é moderno, responsivo e construído sobre a identidade visual da marca (azul escuro `#061630` e laranja `#ff6300`).

---

## 🚀 Tecnologias Utilizadas

- **Angular** (TypeScript)
- **HTML5 / CSS3** (animações customizadas, responsividade)
- **Bootstrap 5.3**
- **Bootstrap Icons 1.11**
- **JavaScript** (vanilla)

---

```
site-reinar/

└── js/
    ├── slides.js            # Lógica do slider principal
    ├── animbouce.js         # Animação bounce ao scroll
    ├── cliqueslide.js       # Controle de clique nos slides
    └── slidesdepoimentos.js # Slider de depoimentos
```

---

## ✨ Funcionalidades

- **Navbar responsiva** com menu hamburguer para mobile
- **Slider de fotos** animado com navegação por botões
- **Seção de serviços** com cards animados (restauração, limpeza, polimento)
- **Seção de depoimentos** com slider próprio
- **Animação bounce** nos elementos ao entrar na tela
- **Footer completo** com contato, serviços, menu e redes sociais
- **Botão de WhatsApp** para contato direto
- **Design totalmente responsivo** (mobile-first)

## 🖥️ Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/jvcod3r/site-reinar.git
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Rode o servidor de desenvolvimento:
   ```bash
   ng serve
   ```

4. Acesse no navegador: `http://localhost:4200`

> Necessário ter o **Node.js** e o **Angular CLI** instalados.
